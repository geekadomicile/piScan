How to install
--------------
	::
		$ cp action.script /usr/share/scanbd/scripts/

Dependencies
------------
	::

		$ sudo su
		$ apt-get update
		$ apt-get install sane-utils
		$ apt-get install imagemagick
		$ apt-get install poppler-utils
		$ apt-get install scanbd
		$ adduser pi lp
		$ scanimage -L
